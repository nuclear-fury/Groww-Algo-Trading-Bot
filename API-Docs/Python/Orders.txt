Title: Orders - Groww API

URL Source: http://groww.in/trade-api/docs/python-sdk/orders

Markdown Content:
This guide describes how to manage orders using the SDK. You can create, modify, cancel, and retrieve order details.

Use the place_order SDK method to register a new order. When successful, the method returns a unique order ID and its initial status.

### [Python SDK Usage](http://groww.in/trade-api/docs/python-sdk/orders#python-sdk-usage)

#### [Request Schema](http://groww.in/trade-api/docs/python-sdk/orders#request-schema)

| Name | Type | Description |
| --- | --- | --- |
| trading_symbol `*` | string | Trading Symbol of the instrument as defined by the exchange |
| quantity `*` | int | Quantity of stocks to order |
| price | float | Price of the stock in rupees case of Limit order |
| trigger_price | float | Trigger price in rupees for the order |
| validity `*` | string | [Validity](https://groww.in/trade-api/docs/python-sdk/annexures#validity) of the order |
| exchange `*` | string | [Stock exchange](https://groww.in/trade-api/docs/python-sdk/annexures#exchange) |
| segment `*` | string | [Segment](https://groww.in/trade-api/docs/python-sdk/annexures#segment) |
| product `*` | string | [Product type](https://groww.in/trade-api/docs/python-sdk/annexures#product) |
| order_type `*` | string | Enumeration of [order types](https://groww.in/trade-api/docs/python-sdk/annexures#order-type) with examples |
| transaction_type `*` | string | [Type](https://groww.in/trade-api/docs/python-sdk/annexures#transaction-type) of the trade |
| order_reference_id | string | User provided 8 to 20 length alphanumeric string with atmost two hypens. (-) |

`*`required parameters

### [Response](http://groww.in/trade-api/docs/python-sdk/orders#response)

#### [Response Schema](http://groww.in/trade-api/docs/python-sdk/orders#response-schema)

| Name | Type | Description |
| --- | --- | --- |
| groww_order_id | string | Order id generated by Groww for an order |
| order_status | string | Current status of the placed order `NEW,ACKED,TRIGGER_PENDING,APPROVED,FAILED,EXECUTED,DELIVERY_AWAITED,CANCELLED, CANCELLATION_REQUESTED, MODIFICATION_REQUESTED,COMPLETED,REJECTED,CANCELLED` |
| order_reference_id | string | User provided reference id to track the status of an order. |
| remark | string | Remark for the order |

You can change or update any pending or open orders using the `modify_order` method.
